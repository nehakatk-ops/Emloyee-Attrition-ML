# Employee Attrition Prediction using Neural Networks

## Project Overview

This project predicts employee attrition using a Neural Network (MLPClassifier) built in Python. The project follows a complete machine learning workflow, beginning with data cleaning and preprocessing, followed by exploratory data analysis, feature engineering, model development, and performance evaluation.

The objective is to identify employees who are likely to leave an organization by analyzing demographic information, job characteristics, compensation, and work-related factors. The project demonstrates the end-to-end process of developing a classification model for HR analytics.

## Objectives

- Clean and prepare employee data for analysis.
- Explore employee characteristics through visualizations.
- Transform and preprocess features for machine learning.
- Train a Neural Network classifier.
- Evaluate model performance using classification metrics.
